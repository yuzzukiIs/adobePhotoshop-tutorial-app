Рабочие инстурменты Adobe Photoshop

Инструменты группы «Выделение»

Selection tools

Прямоугольная область (hotkey: M) — выделяет прямоугольную (квадратную при зажатой клавише shift) область. Овальная область (M) — выделяет овальную (круглую при зажатой клавише shift). Вертикальная строка — выделяет строку пикселей. Горизонтальная строка — выделяет столбец пикселей.

Lasso

Лассо (L) — выделение областей свободной формы. Прямолинейное лассо (L) - предназначено для выделения в изображении областей, ограниченных многоугольниками, составленными из отрезков прямых линий произвольной формы. Магнитное лассо (L) - используется для создания выделений вокруг объектов. Границы области "цепляются" за края объекта, на которых происходит смена цветового тона и насыщенности. (для завершения выделения с помощью лассо необходимо замкнуть область).

Move

Перемещение (V) — используется для перемещение выделенных слоев, объектов, направляющих. При зажатой клавише Shift перемещение будет строго по горизонтали или вертикали.

Magic Wand

Волшебная палочка (в раздвижном меню еще будет «быстрое выделение») (W) - Выделяет области на основе сходства цветов смежных пикселов.

Quick Selection

Инструмент «Быстрое выделение» позволяет быстро «нарисовать» выделенную область с помощью регулируемого круглого кончика кисти.

Инструменты группы «Фрагмент и рамка»

Slice

17. Раскройка (K) — создает ломтик для карт рисунков, используемых в документах HTML. Выделение фрагмента (K) — выделяет ломтик и редактирует его.

Crop

Рамка (С) — вырезает из изображения выделенную область, отсекая ненужные фрагменты.

Slice Select

Инструмент «Выделение фрагмента» (Slice Select) выделяет фрагменты.

Инструменты группы «Ретуширование»

Spot Healing Brush

Точечная восстанавливающая кисть (J) — использует пиксели по образцу изображения или узора и сопоставляет их текстуру, освещение, прозрачность и затенение с соответствующими параметрами исправляемых пикселей.

Healing Brush

Восстанавливающая кисть (J) - использует пикселы по образцу изображения или узора и сопоставляет их текстуру, освещение, прозрачность и затенение с соответствующими параметрами исправляемых пикселов (отличие от точечной восстанавливающей кисти в том, что нужно выделять область из которой будет копироваться пиксели).

Patch

Заплатка (J) - служит для исправления дефектов. Нужно построить выделение донорской области, совпадающее с дефектом по своей геометрии и микроструктуре, и перетащить выделение инструментом на поврежденный фрагмент.

Red Eye

Красные глаза (J) — используется для удаления красных глаз на фотографиях.

Clone Stamp

Штамп (S) — копирует пиксели из одной области в другую.

Pattern Stamp

Узорный штамп (S) — копирует выделенную область как узор.

Eraser

Ластик (E) — закрашивает объекты в цвет предыдущего слоя.

Background Eraser

Фоновый ластик (Е) — заменяет фон изображения прозрачной областью.

Magic Eraser

Волшебный ластик (Е) — удаляет подобные цвета, распологающиеся в зоне кисти.

Blur

Размытие (R) – имитирует одноименный фильтр в фотошопе.

Sharpen

Резкость (R) – имитирует одноименный фильтр в фотошопе.

Smudge

Размазывание (R) — смещает пиксили, имитируя мазки.

Dodge

Осветлитель (О) — осветляет отдельную область изображения (можно использовать для создания виньетки).

Burn

Затемнитель (О) — затемняет отдельную область изображения (можно использовать для создания виньетки).

Sponge

Губка (О) — увеличивает или уменьшает насыщенность отдельной области изображения.

Инструменты группы «Рисование»

Gradient

Градиент (G) — позволяет заливать выделенную область плавным переходом от одного цвета к другому.

Paint Bucket

Заливка (G) — позволяет закрашивать однородные пикселы выделенной области изображения выбранным цветом.

History Brush

Архивная кисть (Y) — рисует мазками, созданными на основе предыдущей версии изображения.

Art History brush

Архивная художественная кисть (Y) — рисует художественными мазками, созданными на основе предыдущей версии изображения.

Brush

Кисть (В) — основной инструмент для рисование, имеет много разновидностей.

Pencil

Карандаш (В) — рисует линии основным цветом с жесткими краями.

Инструменты группы «Рисование и текст»

Path selection

Инструмент «Выделение контура» выделяет фигуры или сегменты, отображая узловые точки, линии направления и точки направления. Выделяет кривую Безье.

Add anchor point

Добавить опорную точку — добавляет новую узловую точку в контуре или кривой Безье.

Convert anchor point

Конвертировать опорную точку — преобразует угловую точку в точку привязки и наоборот (опять же в кривых Безье или контурах).

Delete anchor point

Удалить опорную точку — удаляет выбранную узловую точку в контуре или кривой Безье.

Partial selection

Частичное выделение (А) — применяется для корректировки контура, путем перемещения узловых точек.

Feather

Перо (Р) — используется для рисования кривых Безье и контуров по точкам привязки.

Text

Инструмент «Текст» создает текст на изображении. Более подробно про инструменты для работы с текстом можно прочитать в посте

Type mask

Инструмент «Текст-маска» создает области выделения в форме текста.

Shape

Группа инструментов «Фигура» и инструмент «Линия» рисуют фигуры и линии на обычном слое или слое-фигуре.

Custom Shape

Инструмент «Произвольная фигура» создает заказные фигуры, выбранные из списка заказных фигур.

Horizontal/vertical mask text

Горизонтальный/вертикальный текст-маска (Т) — используется для создания выделенной области в форме символов текста.

Vertical text

Вертикальный текст (Т) — добавляет вертикальный текст на изображение.

Horizontal text

Горизонтальный текст (Т) — добавляет горизонтальный текст на изображение.

Custom Shape

Инструмент «Произвольная фигура» создает заказные фигуры, выбранные из списка заказных фигур.

Line

Линия (U) — рисует линию.

Polygon

Многоугольник (U) — рисует заданный многоугольник.

Ellipse

Эллипс (U) — рисует эллипс.

Rounded Rectangle

Прямоугольник со скругленными углами (U) — рисует прямоугольник со скругленными углами.

Rectangle

Прямоугольник (U) — рисует прямоугольник.

Комментарии, измерения и инструменты навигации

Comments

Комментарии (N) — добавляет на рисунок комментарии и пометки;

Audio

Аудио (N) — добавляет на рисунок звуковые комментарии и пометки.

Eyedropper

Пипетка (I) — позволяет задать основной и фоновый цвета путем взятия проб цвета на изображении или из палитр "Синтез" и "Каталог".

Color standard

Цветовой эталон (I) - служит для размещения в изображении от одной до четырех контрольных точек, цветовые параметры которых отображаются в палитре инфо и тем самым дают, возможность пользователю непрерывно следить за изменением цвета очень важных фрагментов.

Ruler

Линейка (I) — используется для измерений расстояний и углов на изображении.

Hand

Рука (Н) — используется для просмотра частей изображений, который не попали на экран.

Scale

Масштаб (Z) — используется для изменения масштаба в большую или меньшую сторону.
